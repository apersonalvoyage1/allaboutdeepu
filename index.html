📸 How to add your photos

  1. Create a folder called photos next to index.html in your GitHub repo
  2. Upload your images named photo1.jpg through photo6.jpg
  3. For each card, replace <div class="photo-placeholder">…</div> with <img src="photos/photo1.jpg">
  4. Update the caption text in each .photo-caption span
  5. Delete this box before sharing with Deepti 🌸
